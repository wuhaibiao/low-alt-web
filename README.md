# 低空无人机感知系统

## 项目简介

这是一个基于Vue.js开发的低空无人机感知系统前端项目，提供实时的无人机监控、数据可视化和管理功能。

## 功能特性

### 🔐 用户认证系统
- 用户登录/注册功能
- 默认管理员账户：`admin` / `123456`
- 本地存储登录状态

### 📊 数据监控面板
- 实时显示无人机数量统计
- 设备状态分类（在线/飞行/任务/告警）
- 动态数据更新
- 各类型任务统计图表

### 🗺️ 地图可视化
- 基于Leaflet的交互式地图
- 实时显示无人机位置
- 状态标记和信息弹窗
- 深色主题地图样式

### 📈 数据质量监控
- 图像侦察与融合状态
- 传感器健康状态图表
- 实时数据质量指标
- 告警状态监控

### 💻 响应式设计
- 支持多种屏幕尺寸
- 移动端适配
- 现代化UI设计
- 流畅的动画效果

## 技术栈

- **前端框架**: Vue.js 3
- **地图组件**: Leaflet
- **图表库**: Chart.js
- **样式**: CSS3 + 渐变效果
- **图标**: Font Awesome

## 快速开始

1. **克隆项目到本地**
   ```bash
   git clone [repository-url]
   cd 低空大创web
   ```

2. **直接打开网页**
   - 双击 `index.html` 文件
   - 或使用本地服务器打开（推荐）

3. **使用本地服务器**（推荐）
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx serve .
   
   # 使用VSCode Live Server插件
   ```

4. **访问系统**
   - 打开浏览器访问 `http://localhost:8000`
   - 使用默认账户登录：`admin` / `123456`

## 使用说明

### 登录系统
1. 在登录页面输入用户名和密码
2. 默认管理员账户：
   - 用户名：`admin`
   - 密码：`123456`
3. 也可以注册新账户

### 主界面功能
- **左侧面板**：无人机概况统计和设备列表
- **中央区域**：实时地图显示无人机位置
- **右侧面板**：数据质量监控和传感器状态

### 实时功能
- 系统会自动更新时间显示
- 无人机数量会定期刷新
- 地图上的标记显示不同状态的无人机
- 传感器图表实时展示健康状态

## 自定义配置

### 修改默认用户
编辑 `js/app.js` 文件中的用户数据：
```javascript
const users = ref([
    { username: 'admin', password: '123456', email: 'admin@system.com' }
]);
```

### 调整无人机位置
修改 `js/app.js` 中的 `dronePositions` 数组：
```javascript
const dronePositions = [
    { id: 1, lat: 34.7465, lng: 113.6254, name: '巡检无人机001号', status: 'online' },
    // 添加更多位置...
];
```

### 更换地图样式
在 `initMap()` 函数中替换地图图层：
```javascript
L.tileLayer('your-map-tile-url/{z}/{x}/{y}.png', {
    attribution: 'Your attribution'
}).addTo(map);
```

## 开发说明

### 文件说明
- `index.html` - 主页面，包含完整的HTML结构
- `css/style.css` - 主样式文件，包含布局和组件样式
- `css/login.css` - 登录页面专用样式
- `js/app.js` - Vue.js应用逻辑，包含所有交互功能

### 扩展开发
1. 添加新功能组件到 `index.html`
2. 在 `css/style.css` 中添加对应样式
3. 在 `js/app.js` 中实现功能逻辑

## 注意事项

- 本项目为纯前端项目，数据为模拟数据
- 用户信息仅保存在浏览器本地存储中
- 地图需要网络连接加载瓦片图层
- 建议使用现代浏览器以获得最佳体验
